【資料夾】
node_modules => 裝相關套件
public => 放公開可以儲存的資料
src => 開發內容的區域

【檔案】
.gitattributes (不會使用到【github在處理文件會使用到】)
.gitignore (過濾上傳的內容)
index.html (伺服器再啟動會讀取的頁面)
package.json (管理套件的版本資訊)
package-lock.json (本地主機上使用的套件版本)
eslint.config.js (程式碼規則)
vite.config.js (啟動專案的相關資訊)
jsconfig.json (啟動專案的相關資訊)

.editorconfig (排版用的)

README.md (github上顯示的內容)
